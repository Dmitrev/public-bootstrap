# Public bootstrap
The part of my bootstrap of a new machine, which sets up git and authentication.

requires (which should be installed by default on most systems):
- bash
- curl
- apt (if Debian based)
- pacman and yay/paru (if Arch based)

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Dmitrev/public-bootstrap/refs/heads/main/install.sh)
```
